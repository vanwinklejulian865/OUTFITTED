# OUTFITTED

This is a simple static site (HTML/CSS/JS). To publish publicly via GitHub Pages:

1. Make sure Git is installed and your machine is signed into GitHub.
2. Run these commands from your project folder:

```bash
git init
git add .
git commit -m "Initial site"
git branch -M main
git remote add origin https://github.com/vanwinklejulian865/OUTFITTED.git
git push -u origin main
```

3. Open GitHub and go to `https://github.com/vanwinklejulian865/OUTFITTED`.
4. In Settings → Pages, select the `main` branch and `/ (root)` as the source, then Save.
5. Your live URL will be:

`https://vanwinklejulian865.github.io/OUTFITTED/`

6. The files already updated for publishing:
   - `index.html` (meta tags and canonical URL)
   - `sitemap.xml`
   - `robots.txt`

7. Optional: submit the site URL to Google Search Console to request indexing.
